# **App Name**: BudgetWise

## Core Features:

- Expense Tracking: Allow users to manually input expenses with categorization.
- Budget Setting: Enable users to set monthly budgets for various categories.
- Spending Analysis: Visualize spending patterns through interactive charts.
- Dashboard: Display key insights like total spending, remaining budget, and category breakdowns.
- Personalized Budget Advice: Suggest adjustments to budgets based on AI-analyzed spending patterns. Acts as a tool providing advice and pointing out anomalies.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of financial trust and calm.
- Background color: Very light greenish-blue (#E0F8F7). The background should be a gentle variation of the primary.
- Accent color: Yellow-Green (#9ACD32). It's analogous to teal but offers sufficient contrast.
- Body and headline font: 'PT Sans', sans-serif, which offers a good mix of modern look and readability.
- Use simple, clear icons to represent spending categories and actions.
- Design a clean, intuitive dashboard layout with prioritized information.
- Implement subtle animations to provide feedback on user actions.
